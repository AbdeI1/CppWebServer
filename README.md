# C++ Web Server

## Getting Started

### Pre-requisites
 * CMake
 
### Clone

```shell
git clone https://github.com/dtorban/CppWebServer
cd CppWebServer
```

### Build

* Linux and Mac (command line)

  ```shell
  mkdir build
  cd build
  cmake ..
  make
  ```
  
### Run
  
 ```shell
  ./build/bin/ExampleServer 8081
  ```

 * Then open up example/Example.html in a browser (i.e. Firefox).
